## Công nghệ sử dụng (Tech Stack)

- **Frontend:** React (Vite), Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB Atlas
- **Quy trình:** Git-flow, SourceTree

## Quy định dự án
- Mọi tính năng mới phải được triển khai trên nhánh `feature/`.
- Thực hiện Code Review trước khi Merge vào nhánh `develop`.
- Tuân thủ quy tắc viết Commit Message rõ ràng.
